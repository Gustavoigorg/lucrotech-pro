# 💰 LucroTech PRO

**PWA Offline-First** para gestão financeira de assistências técnicas.

## ✨ Funcionalidades
- 📊 Dashboard: Faturamento, lucro real, ticket médio
- 🛠️ Calculadora OS: Mão de obra + margem garantia + rateio custos fixos
- 💾 Dados salvos localmente (localStorage)
- 📱 **Instalável** como app Android/iOS
- 🌐 Funciona **sem internet**

## 🚀 Demo Ao Vivo
[![Demo](https://lucrotechpro.netlify.app/)](https://lucrotechpro.netlify.app/)

## 📱 Instalação
1. Acesse [lucrotechpro.netlify.app](https://lucrotechpro.netlify.app/)
2. Chrome: Menu > "Instalar app"
3. Funciona offline!
